## Prerequisites

You will need NodeJS `v14` to compile this project.


## Local Development Setup

### Install dependencies:
`npm i` or `npm install`

### Start local dev server _(using gulp)_
`npm run dev`

### Open browser to
http://localhost:8000

### Direct links to each page
#### Page-01 LP
http://localhost:8000/page-01.html
